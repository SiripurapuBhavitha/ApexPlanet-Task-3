# ApexPlanet Internship - Task 3

## Project Title
Deep-Dive Analysis & Interactive Dashboard using Power BI

## Objective
To analyze sales data and build an interactive Power BI dashboard with KPIs and visualizations.

## KPIs
- Total Sales
- Total Orders
- Total Customers
- Total Quantity
- Average Order Value

## Dashboard Features
- KPI Cards
- Sales by Category (Bar Chart)
- Sales Distribution by Category (Pie Chart)
- Monthly Sales Trend (Line Chart)
- Category Slicer for interactive filtering

## Key Insights
- Electronics generated the highest sales.
- Sales vary across different product categories.
- Monthly sales trends can be analyzed using the line chart.
- Users can filter the dashboard using the category slicer.

## Tools Used
- Power BI Desktop
